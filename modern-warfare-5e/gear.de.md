# Ausrüstung

Kleidung, Rucksäcke, Feldausrüstung, Kommunikation, Optiken, Elektronik, Sanitätsmaterial, Sprengmittel und Drohnen. Die Effekte der Gegenstände sind vorrangig narrativ -
jeder Gegenstand trägt Tags, die der GM nutzt, um über situativen Vorteil, Zugang oder Fähigkeit zu entscheiden. Gewichtsangaben
sind vorhanden, damit Operators ihre Traglast verfolgen können.

## Ausrüstungs-Tags {#gear-tags}

| Achse     | Tag            | Effekt                                                                                      |
|-----------|----------------|---------------------------------------------------------------------------------------------|
| Vision    | night-vision   | Behandle Dunkelheit/Zwielicht als Taghelligkeit bis zur Geräte-Reichweite (typ. 100 ft).    |
| Vision    | thermal        | Sieht Wärmesignaturen durch Rauch, Laub und leichte Sichtbehinderung.                       |
| Vision    | magnified      | Advantage auf Perception (Sicht) auf lange Distanz; Disadvantage beim Erkennen naher Bewegung. |
| Signal    | encrypted      | Kommunikation kann nicht von unausgerüsteten Parteien abgehört oder gelesen werden.         |
| Signal    | long-range     | Wirksam jenseits der Standard-Gruppenkommunikation (regionale/Satellitenabdeckung).         |
| Signal    | short-range    | Nur Sichtlinie; kleiner Radius (typ. unter 1 km).                                           |
| Signal    | jams-comms     | Unterdrückt feindlichen Funkverkehr in kurzem Radius.                                       |
| Detection | detects-motion | Warnt vor Bewegungen in einem definierten Bereich.                                          |
| Detection | detects-rf     | Erfasst Funk- oder elektronische Emissionen in Reichweite.                                  |
| Utility   | breaching      | Kann Schlösser, Türen, Wände oder ähnliche Hindernisse überwinden.                          |
| Utility   | lockpicks      | Ermöglicht Sleight of Hand / Thieves' Tools Checks gegen mechanische Schlösser.             |
| Utility   | repair         | Ermöglicht Reparatur-Checks an einer bestimmten Ausrüstungsklasse (Drohne, Waffe, Fahrzeug).|
| Medical   | stops-bleed    | Stabilisiert ein blutendes Ziel; interagiert mit Triage Code.                               |
| Medical   | revives        | Bringt ein bewusstloses Ziel nach GM-Ermessen zurück auf Triage Code Gelb.                  |
| Carry     | covert         | Verbirgt den Inhalt vor beiläufiger Kontrolle (zivile Tarnung).                             |
| Carry     | high-capacity  | Fasst deutlich mehr als ein Standard-Rucksack; bringt Gewicht mit sich.                     |

## Missions-Kits {#mission-kits}

All-in-One-Bündel, die die Frage "was ist der Einsatz?" beantworten - wähle eine Zeile passend zum Missionstyp. Die unten verlinkten
Abschnitte zerlegen jedes Kit in Einzelteile.

| Kit                  | Gewicht | Rolle                                   | Inhalt (Zusammenfassung)                                                                   |
|----------------------|---------|-----------------------------------------|--------------------------------------------------------------------------------------------|
| Direct Action        | ~18 kg  | Offensiver Raid / HVT-Aktion            | Hauptrucksack + IFAK + NVG + verschlüsseltes Squad-Funkgerät + Breaching-Ladung + Extra-Magazine |
| Reconnaissance       | ~12 kg  | Langstrecken-Stealth-Beobachtung        | Tagesrucksack + Fernglas + Spotterscope + Funkgerät + GPS + leichtes Überlebenskit         |
| Long-Range Patrol    | ~25 kg  | Mehrtägige Überland-Infiltration        | Voller Rucksack + Sat-Funkgerät + NVG + GPS + Rationen + Überlebenskit + Wasserfilter      |
| Hostage Rescue / CQB | ~15 kg  | Gebäudesäuberung, Breach-and-Rescue     | Chest Rig + IFAK + NVG + Flashbangs + Breaching-Ladung + verschlüsseltes Funkgerät         |
| Protective Detail    | ~6 kg   | Zivilbegleitung / Low-Profile-Sicherheit| Verdeckte Tasche + verdecktes IFAK + Ohrhörer + Kurzstrecken-Funkgerät                     |
| Humanitarian / SERE  | ~10 kg  | Nicht-Kampf oder Escape-and-Evasion     | Tagesrucksack + erweiterte Medizin + Funkgerät + GPS + Überlebenskit                       |

| T&E Kit         | ~2.0 kg | [Machine Gunner](designations/machine-gunner.md)                 | Traverse and Elevation Mechanismus für stativmontierte MGs; Skalenkopf, Klemmhalterung, T&E Platte.                                          |
