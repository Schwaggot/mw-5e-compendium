# Gear

Packs, field kit, comms, optics, electronics, medical, demolitions, and drones. Item effects are primarily narrative -
each item carries tags that the GM uses to rule on situational advantage, access, or capability. Weight values are
provided so Operators can track encumbrance.

## Gear Tags

| Axis      | Tag            | Effect                                                                              |
|-----------|----------------|-------------------------------------------------------------------------------------|
| Vision    | night-vision   | Treat darkness/dim light as bright light out to device range (typ. 100 ft).         |
| Vision    | thermal        | See heat signatures through smoke, foliage, and light concealment.                  |
| Vision    | magnified      | Advantage on Perception (sight) at long range; disadvantage to spot close movement. |
| Signal    | encrypted      | Comms cannot be intercepted or read by unequipped parties.                          |
| Signal    | long-range     | Effective beyond standard squad comms (regional/sat coverage).                      |
| Signal    | short-range    | Line-of-sight only; small radius (typ. under 1 km).                                 |
| Signal    | jams-comms     | Suppresses enemy radio traffic in a short radius.                                   |
| Detection | detects-motion | Alerts to movement in a defined area.                                               |
| Detection | detects-rf     | Picks up radio or electronic emissions in range.                                    |
| Utility   | breaching      | Can defeat locks, doors, walls, or similar obstacles.                               |
| Utility   | lockpicks      | Enables Sleight of Hand / Thieves' Tools checks vs mechanical locks.                |
| Utility   | repair         | Enables repair checks on a specific class of equipment (drone, weapon, vehicle).    |
| Medical   | stops-bleed    | Stabilizes a bleeding target; interacts with Triage Code.                           |
| Medical   | revives        | Brings an unconscious target back to Triage Code Yellow at GM discretion.           |
| Carry     | covert         | Conceals contents from casual inspection (civilian blend).                          |
| Carry     | high-capacity  | Holds substantially more than a standard pack; incurs weight.                       |

## Mission Kits

All-in-one bundles that answer "what's the op?" - grab a row for the mission type. The sections linked below break each
kit into individual pieces.

| Kit                  | Weight | Role                                   | Contents (summary)                                                             |
|----------------------|--------|----------------------------------------|--------------------------------------------------------------------------------|
| Direct Action        | ~18 kg | Offensive raid / HVT action            | Main ruck + IFAK + NVG + encrypted squad radio + breaching charge + extra mags |
| Reconnaissance       | ~12 kg | Long-range stealth observation         | Day pack + binoculars + spotter scope + radio + GPS + light survival kit       |
| Long-Range Patrol    | ~25 kg | Multi-day overland insertion           | Full ruck + sat radio + NVG + GPS + rations + survival kit + water filter      |
| Hostage Rescue / CQB | ~15 kg | Building clear, breach-and-rescue      | Chest rig + IFAK + NVG + flashbangs + breaching charge + encrypted radio       |
| Protective Detail    | ~6 kg  | Civilian escort / low-profile security | Covert pouch + concealable IFAK + earpiece + short-range radio                 |
| Humanitarian / SERE  | ~10 kg | Non-combat or escape-and-evasion       | Day pack + expanded medical + radio + GPS + survival kit                       |
